What is this?

Hi all! Here you'll find an an example of a quick and dirty e-commerce search experience using Algolia and a sample BestBuy dataset.

What's it made of?

The implementation uses:

  • The AlgoliaSearch JS API
  • Algolia instantsearch.js
  • React
  • Bootstrap SASS
  • Grunt as a build tool
